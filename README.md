Приложение путеводитель с актуальными схемами корпусов РАНХиГС.
Разработано для тестовой публикации в RuStore.

## Использующиеся технологии
* Kotlin
* Clean Architecture + MVVM
* Hilt
* Coroutines
* Material 3
